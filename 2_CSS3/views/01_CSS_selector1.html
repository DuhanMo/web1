태그 선택자

HTML 문서 내의 동일한 태그 모두에게 같은 스타일을 적용할 때 사용한다.

p{color:blue;}와 같은 형채로 사용한다.

아이디 선택자

HTML 문서 내의 선택한 아이디에 해당하는 스타일을 별도로 적용할 때 사용하는 선택자

#아이디명{color:green;}

클래스 선택자

HTML 문서 내의 특정 그룹을 짓는 요소들을 하나의 스타일로 적용할 때 사용한다.

.클래스명{color:puple}
