CSS 속성 선택자

기본 선택자 뒤에 []를 사용하여 특성 속성과 해당 속성의 값을 통해 스타일을 설정할 때 사용한다.

        선택자[속성=속성값]{...}형식으로 쓰인다.
        -- 기호 별 의미 --
        =  : 속성의 값이 같을 때
        ~= : 특정 단어를 속성 값으로 가질 때 (띄어쓰기된 이름 기준 구분)
           ex) class~=test --> class="test abc qwer" ...0
                           --> class="test1234"      ...X
        |= : 속성값-로 시작하는 요소를 선택
        ^= : 속성 값이 특정 단어로 시작할 때 
        $= : 속성 값이 특정 단어로 끝날 때   
        *= : 속성 값이 특정 단어를 포함할 때

반응 선택자

사용자의 움직임(마우스 클릭, 마우스를 올려 놓았을 때)을 감지하여 스타일을 변경하는 선택자이다.

        선택자:active{ ... } / 사용자가 마우스를 클릭했을 때
        선택자:hover{ ... }  / 마우스 커서가 해당 요소 위에 올라갔을 때

동위 선택자

동위 선택자는 동급 관계에 있는 형제 요소에 스타일을 설정할 때 사용한다.

        선택자A+선택자B{....} : A 바로 뒤에 있는 B요소를 선택할 때 사용한다.
        선택자A~선택자B{....} : A 뒤의 모든 B요소를 선택할 때 사용한다.
