window 객체

Window 객체는 자바스크립트의 최상위 개체이며 BOM과 DOM으로 나뉜다.

BOM(Browser Object Model) : location 객체, navigator 객체, history 객체, screen 객체

DOM(Document Object Model) : document 객체
