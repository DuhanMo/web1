제이쿼리 개요

제이쿼리란?

2006년 1월에 존 레식에 의해 개발된 JS 경량화 라이브러리로,
복잡한 자바스크립트 구문들을 짧고 쉽게 개발할 수 있도록
여러 기능들을 제공하는 라이브러리이다.

-- JQuery의 장점 --
1. DOM과 관련된 스크립트들을 짧게 구현 가능하다.
2. 일관된 이벤트 연결을 쉽게 구현
3. ajax(비동기통신), 이벤트 처리 등을 폭넓게 지원한다.
4. JQuery와 관련된 확장형 플러그인, 오픈 API를
지원하여 이를 통해 차트,슬라이드,애니매션 구현이 가능하다.

제이쿼리 연결 방법

1. JQuery사이트에서 JS파일을 직접 다운로드 받아 연결하는 방법
2. CDN(Conetent Delivery Network)을 통해 연결하는 방법

$(document).ready()

현재 웹 문서를 로드 한 후 ready함수를 실행한다
해당 내용은 document.onload와 동일하다. $(document).ready()는 문서가 준비가 완료되면 매개변수로 전달된 함수를 실행하라는 의미
