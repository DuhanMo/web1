객체 추가 삭제 메소드

요소 추가 Part1

$(A).appendTo(B) : A를 B요소 안의 마지막 부분에 추가
$(A).prependTo(B) : A를 B요소 안의 제일 앞부분에 추가
$(A).insertAfter(B) : A를 B요소 뒤에 추가
$(A).insertBefore(B) : A를 B요소 앞에 추가

요소 추가 메소드 part2

$(A).append(B) : B를 A요소 안의 마지막 부분에 추가
$(A).prepend(B) : B를 A요소 안의 제일 앞부분에 추가
$(A).after(B) : B를 A요소 뒤에 추가
$(A).before(B) : B를 A요소 앞에 추가

clone() : 지정한 요소를 복사할 때 사용한다.
remove() : 선택된 노드를 dom에서 제거한다.
detach() : remove 함수처럼 선택된 노드를 삭제합니다.
하지만 detach 함수에 의해 제거된 노드를 임시로 보관할 수 있어서, 제거와 삽입을 반복할 때 많이 사용한다.
empty() : 선택 객체의 자식노드를 삭제한다.
선택한 객체를 삭제하는 remove와는 다르게 자식노드를 삭제하기 때문에
선택객체는 남아있습니다. 객체(요소)를 제거할 때 사용한다.
