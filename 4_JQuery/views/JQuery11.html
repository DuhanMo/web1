이벤트 관련 속성

이벤트 정보는 자바 스크립트 내장 객체인 event 객체를 통해 확인해 볼 수 있다.

이벤트 연결 함수

bind() : 현재 존재하는 요소를 이벤트와 연결할 때 사용하는 함수
unbind() : 현재 연결된 이벤트를 제거하는 함수

마우스를 올려보세요!

jQuery3 버전부터 bind/unbind --> on/off 로
권장함수가 변경되었다.
on() : bind() 대신 이벤트를 연결하는 함수
off() : unbind() 대신 이벤트를 제거하는 함수
